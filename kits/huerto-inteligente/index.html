Kit Prime Huerto Inteligente

Monitorea la humedad del suelo y las condiciones ambientales para cuidar tus plantas de interior —ideal para departamentos, oficinas, salas de clases y huertos controlados.

Componentes incluidos

  • ESP32-WROOM-32 (original, Espressif)
  • Sensor de humedad de suelo capacitivo (no corrosivo)
  • Sensor BME280 (temperatura, humedad relativa, presión atmosférica)
  • Relé 5V 1 canal (para control de bombas o válvulas pequeñas)
  • Fuente Mean Well IRM-15-12 (12V/1.25A, certificada)
  • Cables Dupont (macho-hembra, 20 cm)

Uso recomendado: plantas de interior

Este kit está diseñado para ambientes controlados, como:

  • Departamentos y casas (macetas en balcones techados, interiores)
  • Oficinas y espacios de coworking
  • Huertos escolares en salas de clases
  • Invernaderos cubiertos o con techo

No está diseñado para exposición directa a lluvia, rocío prolongado, sol intenso o polvo ambiental.

Código Arduino (ejemplo básico)

Instala las librerías: Adafruit_BME280, Adafruit_Sensor

#include 
#include 
#include 

#define SENSOR_SUELO 34
#define RELE 23

Adafruit_BME280 bme;

void setup() {
  Serial.begin(115200);
  pinMode(RELE, OUTPUT);
  digitalWrite(RELE, HIGH); // Relé activo en bajo

  if (!bme.begin(0x76)) {
    Serial.println("BME280 no encontrado");
    while(1);
  }
}

void loop() {
  float humedadSuelo = analogRead(SENSOR_SUELO);
  float temp = bme.readTemperature();
  float humedadAire = bme.readHumidity();

  Serial.print("Suelo: "); Serial.print(humedadSuelo);
  Serial.print(" | Temp: "); Serial.print(temp);
  Serial.print("°C | Hum. Aire: "); Serial.println(humedadAire);

  // Riego si suelo está seco (<2000 en escala 0-4095)
  if (humedadSuelo < 2000) {
    digitalWrite(RELE, LOW);  // Activa bomba
    delay(10000);             // 10 segundos
    digitalWrite(RELE, HIGH); // Detiene bomba
    delay(3600000);           // Espera 1 hora
  }

  delay(10000);
}

Recomendaciones de seguridad

  • ✅ Use una caja plástica no conductora para alojar la electrónica.
  • ✅ Conecte la fuente Mean Well directamente a un enchufe con toma de tierra.
  • ❌ No exponga la placa ESP32 ni la fuente a la humedad directa.
  • ❌ No use bombas de más de 1A sin un relé de mayor capacidad.
